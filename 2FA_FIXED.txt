TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "404 Not Found"
   → Make sure Tomcat is running
   → Check URL is correct
   → Restart Tomcat

❌ "Database error"
   → Make sure XAMPP MySQL is running (green)
   → Check database 'eventsphere' exists
   → Try the other setup method

❌ "Can't find the code"
   → Look at Tomcat console (black window)
   → Code is printed there
   → Code expires in 5 minutes

❌ "2FA page doesn't show"
   → Make sure you ran one of the 3 setup methods
   → Check admin has 2FA enabled in database
   → Logout and login again
